ref

ref() worksheet references

Read cells, ranges, dynamic spills, and Name manager values from an R formula.

ref() takes one string containing a case-insensitive Excel-style reference and returns the value identified by it.

ref("A1")A single cell in A1 notation
ref("R1C1")A single cell in R1C1 notation
ref("A1:G5")A rectangular range
ref("B8#")The complete dynamic spill anchored at B8
ref("sales")A range or R expression defined in Name manager
References are case-insensitive. For example, ref("a1") and ref("A1") refer to the same cell. A named expression returns its evaluated R value.
