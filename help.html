RGrid help and keyboard cheatsheet

RGrid is a local, reactive browser spreadsheet powered by webR. Formulas use R, while worksheet references are supplied through ref().

Formula essentials

=mean(ref("A1:A10"), na.rm = TRUE)
=ref("D3#")
=ref("R1C1:R8C7")
={library(magrittr); rnorm(100) %>% matrix(ncol = 2)}
=function(x) x + 3
=\(x) x + 3

Names created in Name manager can refer to ranges or hold R values such as functions and lists.

Function help

Place the formula cursor on a function name and press F1. On ref(), RGrid opens a styled information box inside the app and explains its case-insensitive Excel-style references, including A1, R1C1, A1:G5, B8#, and names defined in Name manager, including named expressions. Core R functions open their R manual page, while functions from other packages open the corresponding CRAN reference-manual anchor.

Examples include lm, capture.output, eurodata::importData, or an unqualified function from a package loaded in the formula.

Click the fx icon beside the formula editor to open the R function index.

Import data

Import data accepts Excel, CSV, and TSV files:

  • Every Excel worksheet becomes a new RGrid worksheet.
  • A CSV or TSV file becomes one new worksheet.
  • Only values are imported. Excel formulas are not imported as R formulas.
  • Imported text remains text even when it resembles an R expression.

Object display

Show element names controls vector names, row names, column names, and dimension names.

Double-click a list cell to inspect a collapsible tree. Use Expand all to reveal every nested branch at once. Double-click a plot cell to reveal its plot.

Keyboard

EnterCommit the formula and move down. Alt+EnterInsert a line break in the formula editor. F1Open documentation for the function under the formula cursor. F2Edit the active cell formula. EscCancel formula editing or close the list tree. Ctrl+C / Ctrl+V / Ctrl+XCopy, paste, or cut formulas and values. Ctrl+Z / Ctrl+YUndo or redo. Shift+ArrowExtend the selection by one cell. Ctrl+ArrowJump to the next data-region edge. Ctrl+Shift+ArrowExtend the selection to a data-region edge. Delete / BackspaceClear selected cells. Selected code + ' " ( [ {Wrap selected formula code in the matching pair.

References and selection

Drag across cells, or use Shift with arrow keys, while editing a formula to insert a range reference. Selecting the complete footprint of a dynamic array inserts a spill reference such as ref("D3#").

Toggle A1/R1C1 switches column headings, the name box, and newly inserted formula references between forms such as A1:G8 and R1C1:R8C7. Existing formulas continue to work in either notation.

Plots

Base R, ggplot2, and lattice plots generated by formulas appear in the resizable Plots pane. Use Set plot size to select the final bitmap dimensions.

Storage and export

RGrid autosaves the exact workbook in browser local storage. It can export an executable and restorable RGrid R script, calculated CSV files in a ZIP archive, or a calculated-value-only Excel workbook.

On GitHub Pages, webR uses its PostMessage communication channel because Pages cannot add the cross-origin-isolation headers used by the faster SharedArrayBuffer channel. Calculations still run locally in the browser.
